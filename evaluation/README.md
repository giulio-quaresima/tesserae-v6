# Tesserae V6 Evaluation

This folder contains benchmark evaluations of Tesserae V6's intertextual search capabilities.

## Evaluations

| Folder | Date | Description |
|--------|------|-------------|
| `2026-02-03_v6_default_lemma_test/` | Feb 3-4, 2026 | Default pairwise lemma matching for Latin |

## Adding New Evaluations

Create new folders using the naming convention:
```
YYYY-MM-DD_description/
```

Each evaluation should include:
- `README.md` - Overview and key findings
- `reports/` - Documentation and reports
- `data/` - Benchmark and analysis data
- `scripts/` - Reproduction scripts
